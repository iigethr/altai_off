# Altai Off
Disable anchor href default behaviour.

## Installation

###### NPM
```
npm i altai_off
```

###### Yarn
```
yarn add altai_off
```

###### Ruby on Rails
Add this line to your application.js
```
//= require altai_off/dist/altai_off
```

Set by using the following:
```
$(".altai-off").altaiOff();
```

Settings:
```
## Print log messages in your developer console.
debug: true or false
```
